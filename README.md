# Polk Development Tracker v7

Upload these files to the root of your GitHub Pages repository.

## What's included
- Larger mobile-friendly logo/header
- Dynamic cards from `data/news.json`
- City, category, status, and keyword filters
- Google News RSS no-key test updater
- GitHub Action that runs every 15 minutes

## Test locally on GitHub Pages
Keep Pages set to: Deploy from branch → main → /(root)

## Run updater manually
GitHub → Actions → Update Polk County News → Run workflow
