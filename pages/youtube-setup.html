YouTube Setup

Connect YouTube Headlines

This feature uses the official YouTube Data API.

  1. Upload Version 7 files
    Includes workflow and YouTube display section.
  2. Create a YouTube Data API key
    Google Cloud Console → APIs & Services → YouTube Data API v3.
  3. Add GitHub secret
    Repo → Settings → Secrets and variables → Actions → New repository secret. Name it YOUTUBE_API_KEY.
  4. Run workflow
    Repo → Actions → Fetch Live Data → Run workflow.

The site searches for positive Polk County growth videos: highways, developments, housing, restaurants, hotels, retail, and openings from the last 30 days. Crime and negative breaking news terms are filtered out.
