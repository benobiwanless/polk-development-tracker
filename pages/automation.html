Live Data Setup

This build includes a GitHub Actions workflow at .github/workflows/fetch-live-data.yml.

After uploading the files, go to your GitHub repo → Actions → Fetch Live Data → Run workflow.

The workflow updates data/live-news.json and data/last-updated.json.

Facebook pages are not directly scraped. They remain source links/manual-review sources.
